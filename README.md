# IM Catálogo Inteligente

Bienvenido al repositorio oficial de **IM Catálogo Inteligente**, la plataforma interactiva de **IM Servicios Contables**.

## Objetivo

Desarrollar una experiencia digital moderna para presentar los servicios de IM, facilitar la solicitud de cotizaciones y servir como punto de entrada al ecosistema digital de la empresa.
